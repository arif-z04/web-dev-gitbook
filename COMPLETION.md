## 🚀 How to Use

### View as Static Files
```bash

code /Web-dev-Gitbook/README.md
```

### Deploy with Gitbook
```bash
# Install Gitbook CLI
npm install -g gitbook-cli

# Install dependencies
cd /path/to/Web-dev-Gitbook
gitbook install

# Serve locally
gitbook serve

# Build static site
gitbook build
```

### View on GitHub
- Upload the folder to GitHub
- Enable GitHub Pages
- Access via `username.github.io/Web-dev-Gitbook`
