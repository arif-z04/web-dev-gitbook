# Step-by-Step Implementation Guide

## Step 1: Set Up Your Project (5 minutes)

1. Create a folder called `portfolio`
2. Inside, create these folders:
   - `css/`
   - `js/`
   - `images/`
3. Create these files:
   - `index.html`
   - `css/style.css`
   - `js/script.js`

## Step 2: Create the HTML Structure (15 minutes)

1. Copy the HTML from source-code.md into `index.html`
2. Update the content with your own information
3. Replace placeholder text with real content
4. Test that the HTML is valid

## Step 3: Style with CSS (45 minutes)

1. Copy the CSS from source-code.md into `css/style.css`
2. Customize colors in the `:root` CSS variables
3. Adjust sizing to match your preferences
4. Test responsive design at different breakpoints

## Step 4: Add Interactivity with JavaScript (20 minutes)

1. Copy JavaScript code into `js/script.js`
2. Test mobile menu toggle
3. Test form validation
4. Test smooth scrolling

## Step 5: Add Images (10 minutes)

1. Add images to the `images/` folder:
   - `hero.jpg` - Hero background
   - `profile.jpg` - Your profile photo
   - `project-1.jpg` - Project screenshots

## Step 6: Test and Polish (30 minutes)

### Desktop Testing
- [ ] All content displays correctly
- [ ] Navigation works
- [ ] Forms validate
- [ ] Links work properly

### Mobile Testing
- [ ] Mobile menu opens/closes
- [ ] Content stacks properly
- [ ] Images are responsive
- [ ] Touch targets are large enough

### Accessibility Testing
- [ ] Keyboard navigation works
- [ ] Form labels are associated
- [ ] Color contrast is sufficient
- [ ] Alt text is present

### SEO Testing
- [ ] Page title is descriptive
- [ ] Meta description is present
- [ ] Headings are properly hierarchical
- [ ] Images have alt text

## Troubleshooting

### Menu not toggling
- Check JavaScript file is linked
- Verify element IDs match

### Form not submitting
- Check JavaScript is loaded
- Verify form ID matches JavaScript

### Images not showing
- Check image paths are correct
- Verify images are in `images/` folder

### Styles not applying
- Check CSS file is linked
- Verify selectors match HTML
- Check specificity of CSS rules
